## 像素版 · 第一章

`pixel/` 目录下是一个全新的像素风同人实现：不依赖原版资源文件，
所有像素美术由代码绘制，剧情对白为原创撰写，可完整游玩第一章
（客栈 → 酒剑仙传剑 → 婶婶病倒 → 仙灵岛 → 蛇妖战 → 拜堂得药 → 忘情返乡）。

直接用浏览器打开 `pixel/index.html` 即可游玩。
方向键/WASD 移动，回车/Z 确认，X/Esc 取消。
